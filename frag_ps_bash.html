💙 PowerShell — Scripting Reference WINDOWS / CROSS-PLATFORM ▶
Reference — PowerShell 7+
Cmdlets · Variables · Flow Control · Pipelines · Common Patterns
PowerShell uses Verb-Noun cmdlet naming. Everything is an object. Pipeline passes objects (not text) between commands. Cross-platform: Windows, macOS, Linux (PS7+).
▸ VARIABLES & DATA TYPES
# Variables always start with $ $name = "Alice" $count = 42 $pi = 3.14 $flag = $true # $true / $false / $null # Typed declaration [int]$n = 10 [string]$s = "hello" [bool]$b = $false [array]$arr = 1,2,3 # Arrays $fruits = @("apple", "banana", "cherry") $fruits[0] # → apple $fruits.Count # → 3 $fruits += "date" # append # Hash tables (dictionaries) $user = @{ Name = "Bob"; Age = 30; Admin = $false } $user.Name # → Bob $user["Age"] # → 30 $user.Role = "Dev" # add key
▸ STRING OPERATIONS
# Interpolation (double quotes only) $msg = "Hello, $name!" $msg = "Count is $($arr.Count) items" # expression inside $() # Here-string (multiline) $text = @" Line one Line two — value is $name "@ # Common string methods "hello world".ToUpper() # HELLO WORLD " trim me ".Trim() # "trim me" "a,b,c".Split(",") # @("a","b","c") "hello".Replace("l","r") # herro "hello world" -match "w\w+" # $true; $Matches[0] = "world" "hello" -like "hel*" # $true (wildcard) -join ("a","b","c") # "abc" ("a","b","c") -join "-" # "a-b-c"
▸ FLOW CONTROL
# If / elseif / else if ($x -gt 10) { Write-Host "big" } elseif ($x -eq 10) { Write-Host "ten" } else { Write-Host "small" } # Comparison operators (not symbols) # -eq -ne -gt -ge -lt -le (numbers) # -like -notlike -match -notmatch (strings/regex) # -and -or -not (logical) # -contains -in (arrays) # Foreach loop foreach ($item in $fruits) { Write-Host $item } # For / While for ($i=0; $i -lt 5; $i++) { Write-Host $i } while ($x -gt 0) { $x-- } # Switch switch ($day) { "Mon" { "Start of week" } "Fri" { "End of week" } default { "Midweek" } }
▸ FUNCTIONS & PIPELINE

🟢 Bash Scripting — Debian / Ubuntu Reference SHELL SCRIPTING ▶
Reference — Bash 5+
Variables · Conditionals · Loops · Functions · Common Patterns
Bash scripts start with a shebang: #!/usr/bin/env bash. Make executable with chmod +x script.sh. Prefer [[ ]] over [ ] for tests.
▸ VARIABLES & QUOTING
# Assign — no spaces around = name="Alice" count=42 files=("a.txt" "b.txt" "c.txt") # indexed array # Read variables echo "Hello $name" # interpolation in double quotes echo 'Literal $name' # single quotes = no interpolation echo "Count: ${count}" # braces for clarity/adjacency # Default values echo "${var:-default}" # use default if var unset/empty echo "${var:=default}" # assign default if unset # Special variables $0 # script name $1 $2 # positional args $@ # all args as separate words $# # number of args $? # exit code of last command $$ # PID of current script $! # PID of last background job
▸ CONDITIONALS
# if / elif / else if [[ $x -gt 10 ]]; then echo "big" elif [[ $x -eq 10 ]]; then echo "ten" else echo "small" fi # Integer tests: -eq -ne -gt -ge -lt -le # String tests: == != < > -z (empty) -n (not empty) # File tests: -f (file) -d (dir) -e (exists) -r -w -x if [[ -f "$file" && -r "$file" ]]; then echo "readable file" fi # One-liner shortcuts [[ -d "/tmp/work" ]] || mkdir -p "/tmp/work" # create if missing command -v git &>/dev/null && echo "git found" # check command exists
▸ LOOPS
# for — iterate array for fruit in apple banana cherry; do echo "$fruit" done # for — C-style for ((i=0; i<5; i++)); do echo $i done # for — glob / find for f in /var/log/*.log; do echo "Processing: $f" done # while while read -r line; do echo "Line: $line" done < file.txt # until until [[ $count -ge 5 ]]; do (( count++ )) done
▸ FUNCTIONS
greet() { local name="$1" # local = scoped to function [[ -z "$name" ]] && { echo "Usage: greet <name>" >&2; return 1; } echo "Hello, $name!" } greet "Alice" # Hello, Alice! greet # prints usage, returns 1 # Return values via echo + command substitution get_date() { date +"%Y-%m-%d"; } today=$(get_date) echo "Today: $today"
▸ SCRIPT BEST PRACTICES
#!/usr/bin/env bash set -euo pipefail # -e exit on error, -u error on unset var, -o pipefail IFS=$'\n\t' # safer word splitting # Trap for cleanup cleanup() { rm -f /tmp/work.$$; } trap cleanup EXIT INT TERM # Logging helper log() { echo "[INFO] $*"; } warn() { echo "[WARN] $*" >&2; } die() { echo "[ERROR] $*" >&2; exit 1; } # Require dependencies for cmd in curl jq git; do command -v "$cmd" &>/dev/null || die "$cmd is required" done
