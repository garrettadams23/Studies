🤝 TCP vs UDP — Transport Layer NETWORK+ ▶
Reference — RFC 793 (TCP) · RFC 768 (UDP)
Connection Types · 3-Way Handshake · Flags · Use Cases
TCP — connection-oriented, reliable, ordered, error-checked. UDP — connectionless, low-overhead, best-effort. Choose based on need for reliability vs speed.
▸ COMPARISON
Feature TCP UDP
Connection 3-way handshake (SYN→SYN-ACK→ACK) None — send and forget
Reliability ACK, retransmission, ordering No guarantees
Speed Slower (overhead) Faster (low overhead)
Header size 20–60 bytes 8 bytes
Flow control Yes (sliding window) No
Use cases HTTP, HTTPS, FTP, SSH, SMTP, databases DNS, VoIP, video streaming, DHCP, gaming
▸ TCP 3-WAY HANDSHAKE & FLAGS
# 3-Way Handshake (connection establishment) Client → Server SYN seq=100 Server → Client SYN-ACK seq=200, ack=101 Client → Server ACK seq=101, ack=201 # Connection established ────────────────────────────── # 4-Way Teardown (graceful close) Client → Server FIN Server → Client ACK Server → Client FIN Client → Server ACK # TIME_WAIT state follows # TCP Flags (URG ACK PSH RST SYN FIN) SYN - Synchronize (start connection) ACK - Acknowledge FIN - Finish (graceful close) RST - Reset (abrupt close / port closed) PSH - Push (flush buffer immediately) URG - Urgent pointer valid ECE - ECN echo (congestion notification) CWR - Congestion window reduced # SYN scan (Nmap -sS) — half-open, sends SYN, reads SYN-ACK/RST # SYN-ACK = open port; RST = closed; no reply = filtered

NMAP — Scan Flags, Responses & Common Options
Nmap is the de-facto tool for network discovery and port scanning. Use nmap <SCAN> <TARGET> syntax. Requires root/administrator for raw socket scans (SYN, FIN, XMAS, etc.).
▸ SCAN TYPES
Flag Scan Type Open Response Closed Response Notes
-sT TCP Connect SYN/ACK RST/ACK Full 3-way handshake; logged by target
-sS SYN (Stealth) SYN/ACK RST/ACK Half-open; doesn't complete handshake; less logged
-sF FIN No response RST Works on Linux/Unix; bypasses some firewalls
-sX XMAS Tree No response RST Sets FIN+URG+PSH flags; binary 00101001
-sN NULL No response Varies by OS No flags set; designed for Linux/Unix
-sA ACK RST No response Maps firewall rules; determines filtered/unfiltered
-sI IDLE (Zombie) SYN/ACK RST/ACK Uses spoofed IP; maximum stealth scan
-sU UDP UDP response ICMP unreachable Slower; needed for DNS/DHCP/SNMP discovery
-sn Ping Sweep — — Host discovery only; no port scan
▸ USEFUL NMAP OPTIONS
# Speed / Timing: -T0 (paranoid) → -T5 (insane) nmap -T4 -A 10.5.23.0/24 # Aggressive: OS detect + version + scripts nmap -sS -p 22,80,443 10.5.23.1 # SYN scan specific ports nmap -sV --top-ports 100 target # Version detection on top 100 ports nmap -sC -sV target # Default scripts + version detection nmap -n -Pn -p- target # All 65535 ports, skip DNS/ping nmap --script vuln target # Run vulnerability scripts nmap --script "vuln and safe" target # Safe vuln scripts only nmap -oX scan.xml target # XML output for import to tools nmap -oN scan.txt target # Normal text output nmap -6 target # IPv6 scan

🎟️ Kerberos Authentication Flow 4-STEP PROTOCOL ▶
Protocol — Source: CEH + CISSP Cheat Sheets
Kerberos — Ticket-Based Authentication (port 88)
Kerberos uses symmetric + asymmetric encryption. Default in Active Directory environments. Attacks: Kerberoasting (offline crack service ticket), Pass-the-Ticket (reuse TGT/TGS), Golden Ticket (forged TGT using KRBTGT hash).
KDC
Key Distribution Centre — central trust authority (runs AS + TGS)
AS
Authentication Service — validates identity; issues TGT
TGS
Ticket Granting Service — issues service tickets upon valid TGT
TGT
Ticket Granting Ticket — proof of identity; valid for session
1
Client → KDC (AS)
Client sends AS-REQ with username in cleartext. No password sent yet.
2
KDC (AS) → Client
AS responds with TGT encrypted with KRBTGT hash. Client decrypts AS-REP using their password-derived key. If incorrect password → cannot decrypt → fail.
3
Client → KDC (TGS)
Client presents TGT to TGS, requests service ticket for specific service (TGS-REQ).
4
KDC (TGS) → Client → Service
TGS issues service ticket encrypted with service account's hash. Client presents ticket to target service. Access granted without re-entering password.

Business Continuity & Disaster Recovery Terminology
BCP ensures business functions continue during disruption. DRP is the tactical recovery plan. Always driven by BIA (Business Impact Analysis) results.
▸ CORE METRICS
RTO
Recovery Time Objective — Maximum acceptable time to restore a system after a disaster. "How fast must we recover?"
RPO
Recovery Point Objective — Maximum acceptable data loss measured in time. "How much data can we afford to lose?"
MTTR
Mean Time to Repair/Recover — Average time to restore a failed component or system to operation
MTBF
Mean Time Between Failures — Average time a repairable system runs before the next failure. Higher = more reliable
MTTF
Mean Time to Failure — Average life expectancy of non-repairable components (used for SSDs, HDDs)

Process — Source: CISSP + Net+ Cheat Sheets
Identification → Preservation → Collection → Examination → Analysis → Presentation
Digital forensics produces court-admissible evidence. Evidence must be relevant, complete, sufficient, and reliable. Chain of custody documents every person who handled evidence, when, and what they did.
1
Identification
Identify potential evidence sources: devices, logs, network captures, cloud data. First responder secures and documents the scene before anything is touched.
2
Preservation
Protect evidence from alteration (write blockers). Follow Order of Volatility: CPU registers/cache → RAM → swap → disk → remote logs → paper. Issue Legal Hold.
3
Collection
Create forensic image (bit-for-bit copy) using tools like dd, FTK Imager, EnCase. Hash the image (MD5/SHA-256) immediately for integrity verification. Never work on originals.
4
Examination
Extract relevant data from the forensic image: files, metadata, deleted files (unallocated space), slack space, registry hives, logs, browser history, memory dumps.
5
Analysis
Interpret extracted data to reconstruct events: timeline analysis, correlation with other evidence, identification of IOCs, attribution of actions to users/systems.
6
Presentation / Court
Produce forensics report documenting methodology, findings, and conclusions. Expert witness testimony must be clear to non-technical judges/jury. E-discovery for legal proceedings.

💾 RAID Levels Reference STORAGE REDUNDANCY ▶
Reference — Source: A+ Cheat Sheet
Redundant Array of Independent Disks — Levels 0, 1, 5, 6, 10
RAID provides fault tolerance and/or performance. Does NOT replace backups. Minimum disk counts and failure tolerance differ by level.
Level Name Min Disks Can Lose Characteristic Use Case
RAID 0 Striping 2 0 Data split across disks; combined read/write speed. No redundancy. Scratch disks, video editing (speed > safety)
RAID 1 Mirroring 2 1 Exact copy on each disk. Half usable capacity. Fast reads, slower writes. OS drives, critical single-point protection
RAID 5 Striping + Parity 3 1 Data + parity distributed. (N-1) usable capacity. Rebuild from parity on failure. File servers, NAS; good balance of speed/redundancy
RAID 6 Double Parity 4 2 Like RAID 5 but two parity blocks. (N-2) usable capacity. Slower writes. Large arrays where double failure is a real risk
RAID 10 Stripe of Mirrors 4 1 per mirror Mirrored pairs then striped. 50% usable capacity. Excellent performance + redundancy. Databases, high-performance high-availability workloads
⚠️ RAID is NOT backup. RAID protects against drive failure only — not against ransomware, accidental deletion, corruption, or site disasters. Follow the 3-2-1 rule: 3 copies · 2 different media types · 1 offsite.

▸ LASER PRINTER IMAGING PROCESS (7 STEPS)
1 — Processing
Page data received; printer RIP (Raster Image Processor) converts to bitmap image in memory
2 — Charging
Primary corona wire/charge roller applies uniform negative charge (~-600V) to photosensitive drum
3 — Exposing
Laser beam neutralizes charge on drum where text/images will appear, creating a latent electrostatic image
4 — Developing
Negatively charged toner is attracted to the uncharged (exposed) areas of the drum; visible image forms
5 — Transferring
Transfer corona belt/roller applies positive charge to paper, attracting toner from drum to paper
6 — Fusing
Fuser assembly (heating element + pressure roller) melts toner permanently into paper fibers (~180°C)
7 — Cleaning
Cleaning blade scrapes residual toner off drum; erase lamp neutralizes drum charge for next cycle

* 0 or more a* greedy; a*? lazy
+ 1 or more a+ greedy; a+? lazy

Token Meaning Example

🔗 URL Encoding — Percent Encoding Reference ENCODING REFERENCE ▶
Reference — W3Schools · MDN · Google · Sigma
URL / Percent Encoding — Rules, Characters & Interactive Decoder
URLs can only contain a safe subset of ASCII characters. Any character outside that set must be percent-encoded: replaced with %XX where XX is the two-digit uppercase hex value of the UTF-8 byte. Also called percent-encoding (RFC 3986).
▸ ENCODING RULES
Safe Characters
Never encoded: A–Z a–z 0–9 - _ . ~
Also safe in specific URL parts: ! * ' ( )
Reserved Characters
Have special URL meaning — encode when used as data:
: / ? # [ ] @ ! $ & ' ( ) * + , ; =
Always Encode
Spaces, Unicode, control chars, and anything outside ASCII printable range must always be percent-encoded. Space → %20 (or + in query strings only)
Query String Rule
In application/x-www-form-urlencoded (HTML forms), space becomes +. In all other contexts use %20.

🏛️ Stoicism — The Art of Rational Endurance PHILOSOPHY ▶
Ancient Greek · Roman
Core Doctrines & Daily Practice
Founded ~300 BC by Zeno of Citium. Stoics held that virtue alone is sufficient for happiness and that emotions rooted in false judgements must be overcome through reason.
▸ THE FOUR CARDINAL VIRTUES
Wisdom (Sophia)
Knowing what is good, bad, and indifferent. The foundation of all other virtues.
Justice (Dikaiosyne)
Acting rightly toward others. Social duty; fairness in all dealings.
Courage (Andreia)
Facing hardship, pain, and death without flinching. Moral as well as physical.
Temperance (Sophrosyne)
Self-discipline and moderation. Mastery over appetites and impulses.
▸ CORE PRINCIPLES
Principle Meaning Practice
Dichotomy of Control Some things are "up to us" (prohairesis) — judgements, impulses, desires. Everything else is external and indifferent. Ask "Is this in my control?" before reacting to any event.
Memento Mori "Remember you will die." Mortality awareness as a tool for gratitude and urgency. Morning reflection: treat each day as a gift, not a given.
Amor Fati "Love of fate." Accept and embrace everything that happens — not mere tolerance, but affirmation. Reframe obstacles as opportunities; say "it was meant to be" and mean it.
Negative Visualization (Premeditatio Malorum) Deliberately imagine loss — of health, relationships, possessions — to appreciate what you have. Evening: briefly imagine losing what you value most, then feel gratitude for its presence.
Sympatheia All things are interconnected in one rational whole (the Logos). We are citizens of the universe. When frustrated by others: "They act from ignorance, not malice. I would do the same without better judgement."
▸ KEY FIGURES & TEXTS
Figure Era Key Work Contribution
Marcus Aurelius 121–180 AD Meditations Roman Emperor. Private journal of self-discipline; applied Stoicism in governance.
Epictetus 50–135 AD Enchiridion, Discourses Former slave. Focused on inner freedom regardless of external circumstance.
Seneca 4 BC–65 AD Letters to Lucilius, On the Shortness of Life Statesman and playwright. Practical moral essays on time, wealth, and death.
Zeno of Citium 334–262 BC — Founder of the Stoic school in Athens' painted porch (Stoa Poikile).
▸ DAILY STOIC PRACTICE
AM
Morning Journal
What obstacles might I face today? How will I respond virtuously? Rehearse the day — adversities and all.
MID
Pause & Label
When emotion arises, name it: "I notice anger." The gap between stimulus and response is where virtue lives.
PM
Evening Review
What did I do well? What could I have done better? No self-flagellation — just honest assessment. (Seneca's method)
♟️ Machiavellianism — Power, Realism & Strategy POLITICAL PHILOSOPHY ▶
Niccolò Machiavelli · 1469–1527
The Prince & the Discourses — Realpolitik Framework
Machiavelli separated political science from ethics. His work describes how power actually operates, not how it ideally should. Often misread as endorsing cruelty; actually a clinical observer of political reality.
▸ CORE CONCEPTS
Concept Meaning
Virtù Not "virtue" but prowess — strength, skill, cunning, willpower. The ruler's capacity to shape events through force of character and action.
Fortuna Fortune/chance — the unpredictable half of all outcomes. Virtù can master ~50% of fate; the rest is Fortuna. She favors the bold.
The Lion & the Fox A ruler must be both: the lion to frighten wolves (force), the fox to recognize traps (cunning). Neither alone is sufficient.
Economy of Violence Cruelties must be committed all at once; benefits distributed gradually. Sudden harshness is quickly forgotten; lingering harm festers.
Appearance vs. Reality It is unnecessary to actually possess virtues — it is sufficient to appear to have them. People judge by what they see, not what is.
Love vs. Fear It is safer to be feared than loved if you cannot be both. But avoid being hated — hatred is fatal; fear can be managed.

Play on People's Need to Believe

☸️ Buddhism — The Path to Liberation SPIRITUAL PHILOSOPHY ▶
Siddhartha Gautama · ~563–483 BC
Four Noble Truths · Eightfold Path · Core Concepts
Buddhism is both a religion and a philosophy. Its goal is the cessation of dukkha (suffering) through understanding the nature of mind and reality. Not theistic in the Western sense — the Buddha is a teacher, not a god.
▸ THE FOUR NOBLE TRUTHS (Catvāri Āryasatyāni)
1
Dukkha (Suffering)
Life inherently contains suffering, dissatisfaction, and imperfection. Birth, aging, sickness, death, getting what we don't want, not getting what we want — all are dukkha.
2
Samudāya (Origin)
The origin of suffering is craving (tanha) — for pleasure, for existence, for non-existence. Attachment and aversion are the roots.
3
Nirodha (Cessation)
Suffering can cease. Nirvana is the complete extinguishing of craving — a state of peace, freedom, and liberation.
4
Magga (The Path)
The Noble Eightfold Path is the way to end suffering. Practical and systematic — not metaphysical belief but lived practice.
▸ THE NOBLE EIGHTFOLD PATH
Path Category Meaning
Right View Wisdom Understanding the Four Noble Truths; seeing reality as it is, not as we wish it to be.
Right Intention Wisdom Commitment to non-attachment, non-ill-will, and non-harmfulness.
Right Speech Ethics Abstaining from lying, divisive speech, harsh speech, and idle chatter.
Right Action Ethics Abstaining from killing, stealing, and sexual misconduct.
Right Livelihood Ethics Earning a living that does not cause harm to others or oneself.
Right Effort Meditation Cultivating wholesome states and abandoning unwholesome ones continuously.
Right Mindfulness Meditation Present-moment awareness of body, feelings, mind-states, and phenomena.
Right Concentration Meditation Developing deep meditative absorption (jhanas) as foundation for insight.
▸ CORE CONCEPTS (THE THREE MARKS OF EXISTENCE)
Anicca — Impermanence
All conditioned phenomena are impermanent. Clinging to what is transient is the source of suffering. Release attachment.
Dukkha — Unsatisfactoriness
Even pleasant experiences carry an undercurrent of dissatisfaction because they end. Nothing conditioned can permanently satisfy.
Anatta — Non-Self
The "self" is not a fixed, permanent entity but a changing process of aggregates (skandhas). Liberating when truly understood.
☯️ Taoism — The Way of Nature CHINESE PHILOSOPHY ▶
Lao Tzu · ~6th–4th c. BC
Tao Te Ching — The Classic of the Way and Its Power
The Tao (道) — the Way — is the fundamental, unnamed principle underlying all of reality. It cannot be named or fully grasped, only lived. "The Tao that can be told is not the eternal Tao."
▸ CORE PRINCIPLES
Principle Meaning Application
Wu Wei (無為) "Non-action" or effortless action. Acting in harmony with the natural flow rather than forcing outcomes. Avoid unnecessary struggle. Like water: yield, go around, persist gently.
Pu (樸) — Simplicity The uncarved block. Original nature before social conditioning. Simplicity as strength. Strip away pretense. Return to what is essential and natural.
Te (德) — Virtue/Power Inner virtue or power that arises naturally when aligned with the Tao. Not morality imposed from outside but integrity from within. Be genuinely good, not performatively good.
Yin-Yang (陰陽) All opposites are interdependent and contain each other's seed. Darkness within light, light within darkness. Change is constant. Stop seeing circumstances as purely good or bad. Both are transforming.
Ziran (自然) Self-so, naturalness. Things should be allowed to follow their own nature. Stop trying to control what is inherently wild and self-organizing.
▸ WATER AS TAOIST METAPHOR
Water is the supreme Taoist metaphor. It is:
· Soft yet wears down the hardest rock (persistence)
· Low — it always seeks the lowest point (humility)
· Yielding — flows around obstacles rather than fighting them
· Essential — all life depends on it, yet it does not demand recognition
"Nothing in the world is as soft and yielding as water, yet nothing is better at dissolving the hard and inflexible."
🌑 Existentialism & Absurdism 20TH C. PHILOSOPHY ▶
Sartre · Camus · Kierkegaard · Heidegger
"Existence precedes essence" — Building meaning in a meaningless universe
Existentialism holds that there is no pre-given human nature or purpose. We exist first, then create ourselves through choices. This is both terrifying (radical freedom) and liberating.
▸ KEY CONCEPTS
Concept Thinker Meaning
Radical Freedom Sartre We are "condemned to be free" — there is no escape from choice. Even refusing to choose is a choice. Total responsibility for self.
Bad Faith (Mauvaise Foi) Sartre Pretending you have no choice; adopting a fixed identity to escape freedom. E.g., "I can't help it, that's just who I am."
The Absurd Camus The conflict between humans' need for meaning and the universe's silent indifference. Three responses: physical suicide (wrong), philosophical suicide (religion — bad faith), or revolt (the only authentic response).
Authenticity Heidegger Owning your existence — facing death (Being-toward-death) rather than fleeing into das Man (the "they-self" of conformity).
Leap of Faith Kierkegaard Rational thought cannot resolve the deepest questions. Authentic existence requires a passionate, subjective commitment despite uncertainty.
Thrownness (Geworfenheit) Heidegger We are "thrown" into existence — we did not choose our time, place, language, culture. But we are responsible for what we do with this situation.
▸ THE MYTH OF SISYPHUS — CAMUS
Sisyphus is condemned to roll a boulder up a hill for eternity — it always rolls back down. Camus argues this is the human condition.

The resolution: "One must imagine Sisyphus happy."
Not because the task is meaningful in any cosmic sense, but because the struggle itself fills his heart. Revolt against absurdity — continuing to live, create, love — is the only authentic response.

🔢 Functional Numbers 1 through 9 — What Each Number Means ALL NUMBERS ▶
Reference — Universal Functional Areas
Numbers are consistent across ALL prefix letters
Regardless of whether the prefix is J, G, S, N, or A — the number always means the same functional area. A J3 and an S3 both handle Operations — just at different command levels.
1
Manpower & Personnel
Human resources, strength reporting, assignments, promotions, awards, disciplinary actions, casualty reporting, morale and welfare. The "HR department" of a military headquarters. J1 · G1 · S1 · N1 · A1
2
Intelligence
All-source intelligence: HUMINT, SIGINT, GEOINT, OSINT, MASINT. Threat assessments, enemy order of battle, ISR (Intelligence, Surveillance, Reconnaissance), targeting. J2 · G2 · S2 · N2 · A2
3
Operations
Current and ongoing operations — the main effort of any HQ. Battle rhythm, OPORD execution, fires coordination, battle tracking, exercises, and training management. The largest and busiest staff section. J3 · G3 · S3 · N3 · A3
4
Logistics & Supply
Sustainment: supply (Class I–IX), maintenance, transportation, field services, mortuary affairs, contracting. Ensures the force has what it needs to fight and survive. J4 · G4 · S4 · N4 · A4
5
Plans & Strategy
Future operations planning — from 96 hours out through long-range strategy. Operational plans (OPLANs), concept development, war gaming, campaign design, interagency coordination. Distinct from J3 (current ops). J5 · G5 · S5 · N5 · A5
6
Communications, Signal & IT
All networks, radios, satellite comms, data links, IT infrastructure, cybersecurity, COMSEC (communications security), and information management. The military's IT department. J6 · G6 · S6 · N6 · A6
7
Training & Force Development
Exercise planning, readiness reporting, individual and collective training programs, force generation, leader development. Air Force A7 = installations & mission support. Usage varies by branch. J7 · G7 · A7
8
Financial Management & Resources
Budgeting, resource allocation, comptroller functions, program objective memoranda (POM), contracts oversight, cost analysis, financial reporting to higher commands. J8 · G8 · N8 · A8

Civil-military operations, host-nation relations, NGO/IGO coordination, reconstruction, PSYOP (Psychological Operations) coordination, USAID partnerships, governance support in conflict zones. J9 · G9 · S9 · A9
🔭 Sub-Designators — The Third Digit (e.g., J63) BRANCH LEVEL ▶
Reference — Sub-Sections Within Directorates
Third Digit = Branch Within the Functional Directorate
Large headquarters (combatant commands, service HQs) further divide each directorate into branches. The third digit follows the same 1–9 logic: 1=Manpower, 2=Intel, 3=Ops, 4=Logistics within that function. Smaller units (battalions, brigades) rarely use sub-designators.

⚖️ Branch Comparison — Same Number, Different Context SIDE-BY-SIDE ▶
Cross-Branch Reference
How Each Branch Uses the Same Numbers Differently
The number means the same function, but the scope, size, and tools differ dramatically between a battalion S-section and a combatant command J-directorate.
▸ FUNCTION 1 — PERSONNEL
Code Level Focus & Scope
S1 Battalion (400–1000 personnel) Leave requests, promotions, awards, strength reports, casualty notification for the battalion. Usually 2–4 people.
G1 Division (10,000–20,000 personnel) Same functions but manages policy across multiple brigades. Interfaces with Army HRC (Human Resources Command).
J1 Combatant Command (multi-service, global) Manages manpower for all services assigned to the command; joint manning documents; handles inter-service personnel policy conflicts.
N1 Navy HQ Total force management for all Navy personnel: recruiting, retention, career development, diversity programs.
A1 Headquarters Air Force Manpower & personnel for the entire Air Force: force structure decisions, officer/enlisted development, Air Force Personnel Center (AFPC) oversight.
▸ FUNCTION 3 — OPERATIONS (THE MAIN EFFORT)
Code Level Focus & Scope
S3 Battalion Plans and supervises all battalion training and combat operations. Runs the battalion Tactical Operations Center (TOC). Usually a Major.
G3 Division Synchronizes maneuver, fires, aviation, and combat support across the division. Manages the Division Main CP battle rhythm.
J3 Combatant Command Executes the commander's operational concept across all services globally. Runs the 24/7 Joint Operations Center (JOC). Often a 2-star billet.
N3 Navy Fleet Fleet operational employment; ship deployment scheduling; current fleet operations and readiness.
A3 Air Force Largest Air Force staff directorate. Oversees air operations, flying hour programs, readiness, and airpower employment globally.
▸ FUNCTION 6 — COMMUNICATIONS & IT
Code Level Focus & Scope
S6 Battalion Maintains radios (SINCGARS, SOCOM radios), tactical comms, NIPR/SIPR terminal at battalion level. Often a Warrant Officer or Captain.
G6 Division Division network (Tactical LAN), frequency management, satellite terminals, Signal Battalion oversight.
J6 Combatant Command Enterprise-level network operations. Manages the Joint Information Environment (JIE); classified and unclassified networks; cyber integration with USCYBERCOM.
N6 Navy Navy enterprise networks (NMCI/ONE-Net), shipboard comms systems, spectrum management, Navy COMSEC program.
A6 Air Force AFNET, Air Force IT infrastructure, C2 systems, spectrum, cyberspace capabilities integration into air operations.
