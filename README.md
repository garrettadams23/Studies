# 🧠 CompTIA & Tech Reference — 2026 Edition

A comprehensive, interactive web-based reference for IT professionals, students, and enthusiasts. This repository organizes complex technical concepts, cycles, and diagrams into a searchable, filterable interface.

## 🚀 Live Demo
Simply open `index.html` in any modern web browser.

## 🌟 Key Features
- **Interactive Filtering**: Quickly toggle between domains (Networking, Security, Linux, AI, etc.).
- **Theme Engine**: Seamlessly switch between **Light Mode** and **Dark Mode** with persistent settings.
- **Deep Technical Content**:
  - **Networking**: OSI Model, TCP/IP, Topologies, and Ports.
  - **Security Core**: Cryptography, PKI, Identity, and Access Management.
  - **Threat & Attack**: Kill Chains, Diamond Model, and OWASP Top 10.
  - **Governance & Risk**: NIST RMF, ISO standards, and Risk Matrices.
  - **Linux & Systems**: File systems, permissions (detailed type indicators), and boot sequences.
  - **AI & Machine Learning**: ML pipelines, paradigms, Agentic AI (MCP, Skills, Sub-agents), and Ethics.
  - **Scripting & Web**: Comprehensive guides for Regex, HTML, CSS, JavaScript (ES6 through ES2025), and SQL.
  - **Productivity**: Dedicated shortcuts for Windows, Linux, Excel, Word, and Spotify.

## 🛠️ Built With
- **HTML5**: Semantic structure for accessibility and SEO.
- **Vanilla CSS**: Advanced CSS variables, flexbox, and grid layouts (no external frameworks).
- **Vanilla JavaScript**: Dynamic filtering, accordion logic, and theme management.
- **Google Fonts**: Share Tech Mono & Outfit for a modern technical aesthetic.

## 📂 Project Structure
- `index.html`: Main application entry point.
- `style.css`: All layout and theme styling.
- `script.js`: Interactive logic and UI state management.
- `Img/`: Technical diagrams, study tips infographic, and favicons.
- `plan.md`: Current development roadmap and task tracking.
