Common Vulnerability Scoring System v3.1 — Base Score Metrics
CVSS provides a standardized numerical score (0.0–10.0) for vulnerability severity. The Base Score reflects intrinsic characteristics. Temporal and Environmental scores refine it. Severity bands: None (0.0) · Low (0.1–3.9) · Medium (4.0–6.9) · High (7.0–8.9) · Critical (9.0–10.0)

MetricOptionsDescription
AV — Attack VectorN · A · L · PNetwork (N), Adjacent (A), Local (L), Physical (P) — how accessible is the vulnerability
AC — Attack ComplexityL · HLow (L)=repeatable; High (H)=requires special conditions beyond attacker control
PR — Privileges RequiredN · L · HNone (N), Low (L), High (H) — privilege level needed before exploiting
UI — User InteractionN · RNone (N)=no user action needed; Required (R)=victim must act
S — ScopeU · CUnchanged (U)=same component; Changed (C)=other security scopes affected
C — ConfidentialityN · L · HNone/Low/High — impact on data disclosure
I — IntegrityN · L · HNone/Low/High — impact on data modification
A — AvailabilityN · L · HNone/Low/High — impact on service disruption
